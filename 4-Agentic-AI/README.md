# Agentic AI

- AI agent can make a plan
- AI agent can take a decision (like which tool to call)
- AI agent can take an action (llike calling a tool)
- depending on the feedback it might re trigger action to achieve desired goal

Ex:-
Normal LLM - simply you can talk to it , ask to book you calendar it can't do
LLM + Tools - We can provide access to our calendar via tool and we can have a converstaion with LLM to book our calendar on certain times (Still no decision)
AI Agent - This has access to calendar tool, also we can give it a goal (reject all unimportant meetings ), this will use it's capability to make a decision (which meetings are unimportant) and take the action (cancel the meet  invite)

### Agentic AI is an system where multiple AI agents are working together (each may solve a part of problem) to achieve an goal.
